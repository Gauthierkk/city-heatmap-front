Each ~50 m grid cell is coloured by its distance to the nearest currently-filtered store: red ≤ 50 m → orange → yellow → green → cyan → blue ≥ 500 m. No address is required to see the overlay; opacity is always controllable. Marker clustering is also removed in favour of plain dots whose radius scales with zoom. |

## Implementation notes

- Distance-field overlay: adaptive cells over the city bbox (Paris 50 m ≈ 70 k cells, NYC 125 m ≈ 149 k cells). Stores are bucketed into a 500 m spatial grid; nearest-neighbour search expands ring-by-ring, early-exiting once the ring's minimum possible distance exceeds the current best. Equirectangular distance approximation (m/° at the city's mid-latitude) replaces haversine per cell. Result rendered to a `<canvas>`, exported as a PNG data-URL, and added as a MapLibre `image` source displayed via a `raster` layer.
- The colour ramp is user-configurable from the "Heatmap settings" panel: "Red within" (10–50 m, default 50 m) and "Blue beyond" (100–500 m, default 500 m); intermediate stops scale proportionally between the two. Recompute is debounced 250 ms since the field costs ~100–300 ms and sliders fire continuously while dragging.
- The overlay is clipped to the city's administrative boundary (`npm run fetch-boundary -- <city>` writes `public/data/boundary-<city>.geojson`, Douglas-Peucker-simplified, area sanity-checked per city: Paris ~105 km²; NYC ~1,223 km² because the OSM admin polygon extends into harbour water). Clipping is a `destination-in` composite fill of the boundary path on the field canvas (even-odd rule handles holes), leaving the per-cell compute unchanged. A thin `line` layer outlines the boundary so the clip edge reads as intentional. If the boundary file is missing or invalid, the overlay renders unclipped.
- Nominatim's usage policy disallows true keystroke autocomplete; suggestions are debounced (800 ms, min 3 chars) to stay well under 1 req/s. Fine for local use; revisit before public hosting.
- Type filters call `setData` with a filtered feature collection; the distance-field overlay recomputes on filter or ramp-bound change.
